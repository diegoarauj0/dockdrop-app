## 📚 Visão geral
Este projeto é um app para gerenciar containers docker.

## Pré-requisitos

- Node.js **v24** ou superior.

## Instalar dependencias

```bash
npm install
```

## Desenvolvimento

Inicia o app em modo de desenvolvimento:

```bash
npm run dev
```

## Build

Gera os arquivos de produção e cria o instalador:

```bash
# Para Windows
npm run build:win

# Para Linux
npm run build:linux
```

## 📦 Instalação

Você pode baixar os .deb, .appImage é .exe página de **Releases**:

👉 https://github.com/diegoarauj0/dockdrop-app/releases

- **Linux**
  - `.deb` (Debian / Ubuntu)
  - `.appImage`
- **Windows**
  - `.exe` installer
